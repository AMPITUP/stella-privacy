# Stella — Privacy Policy

Stella is a private, personal-use application operated solely by its
owner. It is not offered to the public and has exactly one user.

**Data collected:** With the owner's explicit OAuth authorization,
Stella accesses health and fitness data (including WHOOP recovery,
strain, and sleep metrics) belonging solely to the owner.

**Use of data:** Data is used exclusively to provide personal health
tracking, trend analysis, and reminders to the owner. It is never
sold, shared, disclosed to third parties, or used for advertising.

**Storage & security:** Data is stored in the owner's private,
access-controlled database and server infrastructure, encrypted in
transit and at rest.

**Data deletion:** The owner may revoke access at any time via WHOOP
account settings, and may delete all stored data at will.
